## Hands On Work

I started by downloading Oracle's Virtual Box which is which is an open-source virtualization software. 
VirtualBox allows me to run multiple operating systems (virtual machines) on my computer without affecting 
my main operating system.. 
Using VirtualBox, I can create and run virtual machines such as Kali Linux, Ubuntu, and Windows.

I used Kali because it is an open-source operating system mainly used for penetration testing and digital forensics. 

I downloaded the Kali Linux ISO image and attached it to a new VirtualBox virtual machine to install the operating system. 
Then, I started the machine and finished setting it up. 

The first command I ran on the terminal of my Kali system was (sudo apt update). 
It refreshes Kali's list of available software and updates.

Then I ran (sudo apt full-upgrade -y); this command downloads and installs 
the latest version of packages including bug fixes and security patches.

Then I was able to learn the basic commands which included:
- pwd (print working directory)
- ls (list)
- ls -la (list hidden files)
- mkdir (make directory)
- cd (change directory)
- touch (create file)
- nano (use Nano text editor)
- cat (display the contents of the file)
